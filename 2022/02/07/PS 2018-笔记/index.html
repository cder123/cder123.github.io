3、 图层

3.1 图层的概念

为方便编辑，将不同的元素分层。

3.2 图层的种类

  • 普通图层
  • 矢量图层
  • 智能对象图层
  • 形状图层
  • 调整图层
  • 背景图层：不可编辑，双击+确定可以转换为普通图层

  • 新建图层：Ctrl + Shift + N 或点击上图的按钮

  • 填充前景色（先选择颜色）：Alt + Delete
  • 填充背景色（先选择颜色）：Ctrl+ Delete

复制图层（2种）：

  • 选中图层，Ctrl+ C，然后Ctrl+ J
  • 选中图层，拖入右下角的新建图层按钮
  • 新建图层组：选中多个图层，Ctrl + G
  • 取消编组：Ctrl + Shift + G

图层顺序：

  • 鼠标拖动
  • 快捷键：
    • 【向上】：Ctrl + ]
    • 【向下】：Ctrl + [
    • 【置顶】：Ctrl + Shift + ]
    • 【置底】：Ctrl + Shift + [
  • 智能对象图层：

  • 矢量图层
  • 文字图层：

  • 调整层：

栅格化（将不可改的图层变得可改）：选中，右键，栅格化

过滤器：

3.3 图层的功能

  • 叠加
  • 显示图层类型
  • 图层蒙版
  • 调整图层
  • 新建组
  • 新建图层
  • 删除图层

4、文件的新建存储与格式

文件操作：

  • 新建文件：Ctrl + N

  • 保存文件：Ctrl + S

  • 关闭文件：Ctrl + W

文件格式：

  • PSD：源文件
  • JPG：无图层的图片
  • PNG：背景透明的图片

图像大小：

  • Ctrl + Alt + I

画布大小：

  • Ctrl + Alt + C

存储为Web所用格式：

  • Ctrl + Shift + Alt + C

6、 选框工具（M）

取消选区：Ctrl + D

种类：

  • 矩形选框（配合 Shift可以得到 正方形）
  • 椭圆选框（配合 Shift可以得到 圆形）
  • 单行选框：选出水平的1像素行
  • 单列选框：选出垂直的1像素列

工具选项栏

  • 选区中添加（或 配合Shift）
  • 减去（或 配合Alt）
  • 交叉

选框羽化：(增加羽化值，可以将矩形选框 ==变得== 圆角矩形，边缘虚化)

选框样式固定比例，固定大小：

7、套索工具（L）

套索工具的运用：随鼠标滑动而选取

套索的种类：

  • 套索工具
  • 多边形套索工具（鼠标点选，产生定点，最后闭合）
  • 磁性套索工具（沿着物体边缘选取）

套索的功能：（与选框工具一样，可加选、减选、交叉、羽化）

  • 选区中添加（或 配合Shift）
  • 减去（或 配合Shift）
  • 交叉

8、魔棒工具（W）

种类：

  • 魔棒工具（依靠容差值，选取颜色相近的区域）
  • 快速选择工具（类似画笔，拖动鼠标即可选取，画笔大小：[缩小，]增大）

反选：Ctrl + Shift + I

工具选项栏：（与套索工具类似）

  • 加选（或 配合Shift）
  • 减选（或 配合Alt）
  • 交叉

==容差值越小，范围越精确。==

调整边缘，勾选智能半径调整头发边缘效果。

9、裁切工具（C）

裁切后，按Enter确认

种类：

  • 裁切工具
  • 透视裁切工具（点击后，出现网格点，调整网格点的位置，按回车，即可将倾斜的图片变正）
  • 切片工具
  • 切片选择工具

工具选项栏：

  • 比例裁切
  • 拉直（向某个方向鼠标划直线，则向镜面方向转动图片）
  • 参考网格线
  • 其他裁切选项

10、吸管工具（I）

颜色取样器工具：获取颜色的具体数值

标尺工具：（拉动2个标志）量出长、宽、高

工具选项栏：

  • 取样大小
  • 样本

11、修复画笔工具（J）

修补工具：选择区域，拖动选区，自动修补

12、画笔工具（B）

缩小、放大画笔：

  • [
  • ]

颜色替换工具：选颜色，鼠标划动。

钢笔工具（P）

合并两个图层：Ctrl + E

  • 方点：可变化

  • 圆点：划路径

取消手柄：Alt + 单击

移动点：Ctrl + 鼠标拖拽

锚点：在路径上单击创建锚点，鼠标拖拽进行控制

完成路径后，选区：Ctrl + Enter
